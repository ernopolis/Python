8. Para que servem os operadores lógicos and, or e not? Dê um exemplo de situação em que cada um seria usado.

'and' necessita que todos os argumentos sejam verdade:
	if x=1 and y=2:
		print(f'x={x} e y={y})
'or' necessita que pelo menos um argumento seja verdade:
	if x=1 or x=2:
		print(f'x={x}')
'not' inverte o verdadeiro valor.

12. Qual a diferença entre uma variável e uma constante? Python possui constantes nativas?

Variável é um valor que você define que pode mudar durante o programa. Mas é um conceito matemático, normalmente um valor ainda não definido mas que você ainda consegue usar. E constante é um valor imutável, como pi.
Python não tem constantes nativas (que travam o valor).
Mas há uma convenção de se usar letras maiúsculas para indicar que algo é constante.
